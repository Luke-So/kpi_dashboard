# KPI / Performance Dashboard

## Overview
A brief data analysis project exploring business performance using a Superstore dataset.  
The dashboard highlights and identifies performance patterns across time, sales regions, product categories, and discount levels.

## Tools Used
- **Python** (pandas, matplotlib)  
- **Jupyter Notebook**  
- **Excel / Kaggle Superstore Dataset**

## Key Visuals
| Chart | Description |
|--------|--------------|
| **Monthly Sales Trend** | Sales growth over time |
| **Sales & Profit by Category** | Compare product performance |
| **Sales by Discount** | Analyse the effect of discounts |
| **Top 10 Products** | Identify best-sellers |
| **Profit vs. Sales (Coloured by Discount)** | Explore profitability by order |

## Insights
- Highest sales occur in Q4 across multiple years  
- Technology is the most profitable product category  
- Larger discounts tend to lower profitability
